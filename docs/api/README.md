### Audio Analysis Engine

#### AudioAnalyzer

**Klasse**: `src.audio_analysis.analyzer.AudioAnalyzer`

**Beschreibung**: Haupt-Klasse f√ºr Audio-Analyse mit Unterst√ºtzung f√ºr verschiedene Algorithmen.

**Initialisierung**:

**Parameter**:
- `use_essentia` (bool): Essentia f√ºr erweiterte Analyse verwenden
- `use_librosa` (bool): librosa f√ºr Basis-Analyse verwenden
- `cache_enabled` (bool): Cache-System aktivieren
- `quality` (str): Analyse-Qualit√§t ('low', 'medium', 'high', 'ultra')

**Methoden**:

##### `analyze_file(file_path: str) -> Dict[str, Any]`

Analysiert eine einzelne Audio-Datei.

**Parameter**:
- `file_path` (str): Pfad zur Audio-Datei

**R√ºckgabe**:
```python

    'bpm': 128.0,
    'key': 'Am',
    'camelot': '8A',
    'energy': 75.5,
    'mood': 'energetic',
    'duration': 245.3,
    'confidence': {
        'bpm': 0.95,
        'key': 0.87,
        'energy': 0.92,

        'chroma': [...],

        'rms_energy': 0.15
    }
}
```

**Beispiel**:
```python

```

##### `batch_analyze(file_paths: List[str], callback=None) -> List[Dict[str, Any]]`

Analysiert mehrere Audio-Dateien parallel.

**Parameter**:
- `file_paths` (List[str]): Liste von Dateipfaden
- `callback` (callable, optional): Fortschritts-Callback

**Callback-Signatur**:
```python

    print(f"Fortschritt: {current}/{total} - {file_path}")
```

**Beispiel**:
```python

results = analyzer.batch_analyze(files, callback=progress_callback)
```
